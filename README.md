# A shop app with a chat bot sprinkle

## Dependencies
For this app you will need pip (included in [python](https://www.python.org/)) and [npm](https://www.npmjs.com)

## Get Started
Install:
```
npm install
pip install chatterbot flask flask-cors
```
Build the app:
```
npm run build
```
Serve the app:
```
npm run serve
```
